# FinView

微博/雪球财经内容 AI 解读 Chrome 扩展。

## 功能

- 选中文字自动弹出 AI 解读按钮
- 基于通义千问智能分析
- 仅在 weibo.com 和 xueqiu.com 生效

## 安装

1. 下载本项目
2. Chrome 访问 `chrome://extensions/`
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」选择本项目
5. 点击扩展图标输入 API Key

## 配置

点击扩展图标，输入 [阿里云 DashScope](https://dashscope.console.aliyun.com/) 的 API Key。
